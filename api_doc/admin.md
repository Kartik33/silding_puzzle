**`/player`**
----

Returns a list of players

* **Method:** `GET`

* **Success Response:**
  * **Code:** 200 <br />
    **Content:** `{ players: players}`
    
    
**`/player/<int:player_id>`**
----

Deletes the player with player_id

* **Method:** `DELETE`

* **Success Response:**
  * **Code:** 200 <br />
    **Content:** `{ player_id : 12 }`
